Launch Web App

🚀 Quick Start

  1. For Live Server: Set the root directory to apps/web
  2. For Development: Run npm run dev:web from the root
  3. For Production: Run npm run build:web then npm run start:local

🔧 Live Server Configuration

Problem: If apps/web/index.html is not opening, Live Server root is set incorrectly.

Solution: Right-click on the apps/web folder and select "Open with Live Server"
